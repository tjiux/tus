# Tus - 北京工业大学试卷共享平台 📚

> 分享历年期中期末试题，帮同学们轻松备考！

## 访问地址
**[https://tjiux.github.io/tus/](https://tjiux.github.io/tus/)**

## 功能
- 📖 按科目浏览试卷
- 🔍 按年份、学期筛选
- 📥 下载 PDF 试卷
- ➕ 管理员可添加科目和上传试卷

## 技术栈
- 纯静态网站 (HTML/CSS/JS)
- Tailwind CSS 样式
- GitHub Pages 托管
- GitHub Actions 自动部署

## 本地开发
```bash
# 克隆项目
git clone https://github.com/tjiux/tus.git

# 本地预览
python -m http.server 8000

# 管理内容
node scripts/manage.js
```
